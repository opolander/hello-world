.gitignore is a list of of files that Git should not track (files with passwords)

license file is the type of license put on the project
	-MIT License: Simple. Permissive. Lets people do what they want.
	-Apache License 2.0: Permissive like MIT but provides an express grant of patent rights from contributors to users.
	-GNU GPLv3: Copyleft license that requires derivative work and distributors to be held under the same terms. Patents.
	-No license: Default copyright laws apply. Retain all rights.
